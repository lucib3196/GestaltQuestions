Evaluate the following expressions using the order of operations:
1. $7 + 3 \times (10 - 4)$
2. Compare $40 / 8 \times 2$ and $40 / (8 \times 2)$
3. Simplify $5^2 - (2 + 3) \times (7 - 2) + 8 / 2$

Solution for Question 1: Simplifying Expressions

First, we simplify the expression inside the parentheses: $$10 - 4 = 6$$ Next, we perform the multiplication: $$3 \times 6 = 18$$ Finally, we complete the addition: $$7 + 18 = 25$$

Solution for Question 2: Comparing Different Operations

For the expression $40 / 8 \times 2$, we perform the division first: $$40 / 8 = 5$$ Then we multiply the result by 2: $$5 \times 2 = 10$$ For the expression $40 / (8 \times 2)$, we first solve inside the parentheses: $$8 \times 2 = 16$$ Next, we perform the division: $$40 / 16 = 2.5$$

Solution for Question 3: Evaluating a Complex Expression

First, evaluate the exponent: $$5^2 = 25$$ Next, solve the operations inside the parentheses: $$(2 + 3) = 5$$ and $$(7 - 2) = 5$$ Multiply the results from the parentheses: $$5 \times 5 = 25$$ Next, divide: $$8 / 2 = 4$$ Finally, calculate the total result: $$25 - 25 + 4 = 4$$
